▶ 특강 진행 순서

1. 과제 수행관련 애로사항 논의

  - 주제 관련
  - 파이썬 Spyder 설치
  - 파이썬 코드
  - 과제 수행 => 네이버 데이터 수집 / 감성분석 및 시각화

2. 보고서 작성 관련

- 주제 : 관심 주제 선정
   - 데이터수집 : 네이버 Open API 기반 뉴스 데이터 수집
   - 분석 : 한글 텍스트 분석, 감성분석모델 구축, 수집 데이터의 감성분석
   
    제목 : 감성분석모델 기반의 OOO 데이터분석 
    목차 : 
          1. 서 론(1페이지 내외)
          2. 선행연구(2페이지 내외)
          3. 감성분석모델 (5 페이지 내외?)
          4. 데이터분석(5 페이지 내외?)
          5. 결론(1페이지 내외)
          

    내용: 
         1. 서론 => 배경, 필요성(분석 필요성), 목적 => 1페이지 내외
         2. 선행연구 
                - 텍스트마이닝 개요
                - 감성분석 개요
                - 관련 산업 동향
                    - 빅카인즈
                    - 구글트렌드
                    - 섬트렌드 
                - 기계학습 및 회귀분석 개요

         3. 감성분석모델 구축
             - 네이버 영화평 데이터와 로지스틱 회귀분석을 이용한 감성분석모델 구축과정 설명
             - 알고리즘 형태로 설명(그림으로)
                  - 영화평 데이터 수집 => 데이터 전처리 => 벡터 데이터 생성 => 로지스틱회귀모델 도출 => 테스트 등
         4. 데이터분석
              - 관심분야의 텍스트 데이터 수집 소개
              - 텍스트 데이터의 구조
              - 분석결과(긍정/부정 결과, 빈발출현 단어의 시각화 등)
              - 시사점 서술 
              
         5. 결론
             - 연구결과 요약(장점 위주)
             - 기대효과(산업, 학문 등에 어떤 부분 기여할 수 있는지 등)
             - 한계점( 단점 서술 및 보완 방안) 

         6. 참고문헌
