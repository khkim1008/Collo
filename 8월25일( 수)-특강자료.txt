 2. SW관련 분야 진학 탐색
     - 관련 학과 분석(컴퓨터공학, 전산통계, 경영정보, 인공지능학과)

     - 컴퓨터공학과

       => 컴퓨터의 HW와 SW분야 학습, 추가로 약간의 수학
       => HW분야 : 논리회로, 컴퓨터구조, 정보통신
       => SW분야 : 프로그래밍, 자료구조, 알고리즘, 데이터베이스, SW공학, 그래픽
       => 수학 : 통계, 이산수학, 선형대수, 수치해석 등

     - 전산통계학과 => 데이터사이언스학과로 전환 예정

       => 컴퓨터 관련 분야 + 통계학(수리통계, 이론통계 등)

     - 경영정보학과 
       => 컴퓨터 관련분야 + 경영분야(마케팅, 경영분석, 전자상거래, 창업 등)
    
     - 인공지능학과
       => 컴퓨터 관련 분야 + 인공지능 수학
